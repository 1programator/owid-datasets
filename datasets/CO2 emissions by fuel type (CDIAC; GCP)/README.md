# CO2 emissions by fuel type (CDIAC; GCP)

Recent data (from 1959 onwards) is sourced from the Global Carbon Budget (2018). Earlier and regional data is source from CDIAC. 

Data was originally reported in units of carbon. This has been converted to units of CO2 using a conversion factor of 3.664.

Full CDIAC reference:
T.A. Boden, G. Marland, and R.J. Andres. 2017. Global, Regional, and National Fossil-Fuel CO2 Emissions. Available at: doi:10.3334/CDIAC/00001_V2010
